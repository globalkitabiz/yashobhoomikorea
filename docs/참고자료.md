## 3. 7대 세션별 상세 (좌장 기업 + 참여 기업)

### 세션 I: AI+반도체 (AI + Semiconductor)
- **좌장 기업**: SK실트론/무루가파, 아다니, 베벤타
- **참여 기업**: 네이버 클라우드, 리벨리온, 가온칩스, Tessolve(인도), 케이엔에스(K-MSME)
- **논의 주제**: ISM 2.0 반도체 특화 구역, 온디바이스 AI 생산 기지, 28nm→12nm 기술 전이

### 세션 II: 방산 (Defense)
- **좌장 기업**: 한화에어로스페이스, 국방기술진흥연구소, 코비코 / 무루가파(CUMI), L&T
- **참여 기업**: 현대로템, KAI, 풍산, LIG넥스원, 마힌드라 디펜스(인도), SNT모티브(K-MSME)
- **논의 주제**: DAP 2026 공동개발, K9 국산화율 60%+, AI MUM-T R&D

### 세션 III: 모빌리티 (Mobility)
- **좌장 기업**: 현대자동차, KGM / 타타 상용차, 올라
- **참여 기업**: LG에너지솔루션, 기아, 현대모비스, 삼성 SDI, 성우하이텍(K-MSME)
- **논의 주제**: 역관세 구조 개선, 수소 SUV, BMS 현지화, LFP 배터리 생산

### 세션 IV: 에너지 (Energy)
- **좌장 기업**: SK E&S / 두산에너빌리티, 포스코홀딩스 / 릴라이언스 인더스트리
- **참여 기업**: 효성중공업, 한화솔루션, JSW Steel(인도), Adani Green(인도), 일진하이솔루스(K-MSME)
- **논의 주제**: 그린 수소 전력망 이용료 면제, CBAM 공동 탄소 인증, HyREX 실증

### 세션 V: 케미컬 (Chemicals)
- **좌장 기업**: LG화학, 여수산단 / 아다니, 릴라이언스
- **참여 기업**: 롯데케미칼, 한화케미칼, 금호석유화학, 아디티야 비를라 그룹(인도), 국도화학(K-MSME), 솔브레인
- **논의 주제**: PCPIR 한국 전용 클러스터, 양극재 인도 생산, 폐플라스틱 재활용

### 세션 VI: 엔터테인먼트 (Entertainment)
- **좌장 기업**: 하이브 뭄바이 / JioStar (Reliance/Disney)
- **참여 기업**: 스튜디오드래곤, 넷마블, Zee TV(인도), Sony Pictures India(인도), 덱스터 스튜디오(K-MSME)
- **논의 주제**: 공동 제작 쿼터 인정, K-드라마 리메이크, AI 더빙/자막 기술

### 세션 VII: 바이오메디컬 (Bio-medical)
- **좌장 기업**: Seers, SD바이오센서 / 바이오콘
- **참여 기업**: 셀트리온, 삼성바이오로직스, 씨젠, 오스템임플란트, Dr. Reddy's(인도), 바텍(K-MSME)
- **논의 주제**: 의료기기 PLI 확대, CDSCO 임상 상호 인정, GLP-1 펩타이드 공동 개발
